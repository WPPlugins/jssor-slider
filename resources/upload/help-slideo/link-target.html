Link Target [Optional]
Specifies where to open the link, default value is _self.
If the Link Url property is not specified, this property will be ignored.
Posible values are _blank, _self, _parent, _top or framename.

_blank: Opens the link in a new window or tab.
_self: Opens the link in the same frame as it was clicked (this is default)
_parent: Opens the link in the parent frame
_top: Opens the link in the full body of the window
framename: Opens the link in a custom named frame

Note the equivalent html code looks like <a target="_blank|_self|_parent|_top|framename" >
